# ExtensionSorter
## Назначение
Программа `ExtensionSorter` предназначена для просмотра статистики файлов по разным расширениям, или же для их перемещения по соответствующим подпапкам.

## Примеры использования
### Общие сведенья
Дла работы программы необходимо указать операцию:
- `stat` — Для просмотра статистики.
- `move` — Для перемещения файлов по подпапкам в соответствии с их расширениями.

Также необходимо указать расположение, где будет осуществляться сбор статистики или же сортировка.

В качестве необязательного флага можно использовать `/rmdir` после обязательных аргументов для удаления пустых папок в расположении.

### -stat
Для примера возьму папку с мемами, где есть файлы разных форматов:
```CMD
Z:\1_downloads\Test>tree /F
Z:.
├───1
│       132_memes.mp4
│       ...
│       131_memes.mp4
│
└───2 часть
    │   102_memes.jpg
    │   ...
    │   127_memes.mp4
    │
    └───Ещё одна
            117_memes.mp4
            ...
            116_memes.jpg
```

Вызов программы:
```CMD
Z:\1_downloads\Test>cd ..
:: Возврат на уровень выше

::          Вызов ↓      ↓ Операция
Z:\1_downloads>extsort stat Test
::               Расположение ↑

:: Вывод программы:
Расширение ".jpeg": 1
Расширение ".jpg": 82
Расширение ".mov": 1
Расширение ".mp4": 65
Расширение ".png": 1
    Всего: 150
```

Применение операции `stat` безопасно, т.к. не вносит никакие изменения в файловую систему.

### -move
Вызов программы:
```CMD
::                       ↓ Другая операция
Z:\1_downloads>extsort move Test
```
После успешной сортировки будет соответствующее уведомление.


Изменения в файловой системе:
```CMD
:: Перейдём в папку, которая была отсортирована
Z:\1_downloads>cd Test

:: И проверим её содержимое
Z:\1_downloads\Test>tree /F
Z:.
├───1
├───2 часть
│   └───Ещё одна
├───.mp4
│       132_memes.mp4
│       ...
│       127_memes.mp4
│
├───.jpg
│       135_memes.jpg
│       ...
│       115_memes.jpg
│
├───.jpeg
│       46_memes.jpeg
│
├───.mov
│       75_memes.MOV
│
└───.png
        101_memes.png
```
После программы файлы были отсортированы по папкам, в соответствии со своим расширением, но пустые папки в расположении остались.

Чтобы пустых папок после работы приложения не оставалось, необходимо указать флаг `/rmdir`.
```CMD
::    Флаг для удаления пустых папок ↓
Z:\1_downloads>extsort move Test /rmdir
```

## Внимание!
Использование данной программы в системных расположениях может повлечь необратимые последствия для операционной системы. Рекомендуется указывать небольшие папки, содержимое которых не имеет особой важности в работе других программ.
